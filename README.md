# 🍽️ Gnocchi & Giggles

A responsive restaurant menu website designed for an Italian-themed restaurant. The website showcases a clean and modern interface with categorized menu items, restaurant information, opening hours, and contact details.

---

## 📖 Project Overview

Gnocchi & Giggles is a single-page restaurant website built using HTML, CSS, and JavaScript. The project focuses on creating an attractive and responsive user interface while providing an easy-to-navigate digital menu experience.

---

## ✨ Features

- 🍕 Interactive menu categories (Pizza, Pasta, Starters)
- 📱 Responsive design for different screen sizes
- 🎨 Modern and visually appealing UI
- 🏠 About section
- 🕒 Opening hours
- 📞 Contact information
- ⚡ Smooth navigation between sections

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- W3.CSS Framework
- Google Fonts
